# Premium Calculator

A satirical **Premium Calculator** that demands a fake payment and optional tip before revealing your answer. Oh, and the math is intentionally wrong.

## Features

- Looks like a normal, clean calculator
- Press **=** → unlock answer for $0.99
- Fake credit card form (accepts literally anything)
- "Purchase Successful" + tip screen
- Every calculation returns a **wrong** number (close, but intentionally off)
- Fully responsive / mobile-friendly
